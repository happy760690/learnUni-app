# uni-app Vue3 + TypeScript 实战项目
- 这是一个基于 uni-app + Vue3 + TypeScript 的跨端项目
- 支持登录鉴权、列表详情流转
- 使用 Mock 数据开发，后期可无缝切换真实接口
- 并引入 uts 封装平台能力

项目实现细节：

> - 我在项目中使用 uts 封装平台能力,页面只通过 TS 工具函数调用,保证业务代码不感知平台差异,后续可以把性能敏感逻辑下沉到 uts。
> - 对于系统类静态能力,我会在 uts 层做缓存,避免重复调用系统 API,并在 TS 层提供兜底，保证页面稳定。
> - 在项目中我使用 uts 封装异步原生能力,并统一 Promise 接口,页面只通过 TS hook 调用,同时支持缓存和异常兜底。
> - 我在项目中通过 uts 封装权限检测,页面只做授权引导,权限状态支持缓存，避免重复判断。
> - 我在项目中设计了统一的应用初始化流程,在 App 启动时并行获取系统、网络和权限状态,并缓存为全局单例,页面只消费初始化结果，不关心底层细节。
> - 我在项目中通过 uts 封装相机能力,提供 Promise 接口和强类型返回,页面只处理业务结果,并统一在 TS 层做异常兜底
> - 在项目中我将文件作为业务对象建模,上传流程通过状态流转管理,当前使用 Mock 上传，后续可无缝切换真实接口,页面只消费最终结果，不关心实现细节
