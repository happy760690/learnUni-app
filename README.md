# uni-app Vue3 + TypeScript 实战项目
- 这是一个基于 uni-app + Vue3 + TypeScript 的跨端项目
- 支持登录鉴权、列表详情流转
- 使用 Mock 数据开发，后期可无缝切换真实接口
- 并引入 uts 封装平台能力

## 技术栈
- uni-app
- Vue 3
- TypeScript
- Pinia
- uts

## 项目特点
- 强类型约束（types / api / 页面分离）
- Mock 数据开发，接口可切换
- 登录态管理 + 权限拦截
- 跨端能力封装（uts）
- 页面逻辑简洁、职责清晰

## 可运行平台
- 微信小程序
- H5
